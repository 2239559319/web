# 体测预约脚本

## 使用方法  
----------------
 1. 复制choose.js中的内容  ![][1]
 2. 在开放系统前登录，在转到[这个页面][2]（就是场次的页面），按下F12找到控制台console
 3. 粘贴到控制台,将文本倒数第二行的数字换为自己要选的场次号,然后回车![此处输入图片的描述][3]
 4. 等待，开放系统时会自动选并跳转选择结果
